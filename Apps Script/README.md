# 📅 GCal Event Generator

Responsive Google Calendar Event Generator built with Google Apps Script.

---

## ✨ Features

- Create Google Calendar Event
- Reminder Notification
- Email Invitation
- Full Day Event
- Repeat Event
- Custom Recurrence
- Responsive Design
- Mobile Friendly
- Automatically Save Data to Google Sheets
